### Invalid Data Examples
```typescript
// Wrong type for summary
{ summary: 123, keyPoints: [], viewpoints: [], bestPractices: [] }
// → ZodError: Expected string, received number

// Missing required field
{ keyPoints: [], viewpoints: [], bestPractices: [] }
// → ZodError: Required

// Wrong array type
{ summary: "Test", keyPoints: "not-array", viewpoints: [], bestPractices: [] }
// → ZodError: Expected array, received string

// Missing nested field
{ summary: "Test", keyPoints: [], viewpoints: [{ perspective: "Author" }], bestPractices: [] }
// → ZodError: Required (for stance field)
```
