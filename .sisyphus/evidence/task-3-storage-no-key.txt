# Task 3: getApiKey No Key Evidence

## Test: getApiKey returns null when no key saved

### Case 1: Empty storage
```typescript
await fakeBrowser.storage.local.clear();
const apiKey = await getApiKey();
expect(apiKey).toBeNull(); // ✓ PASSED
```

### Case 2: Empty string stored
```typescript
await fakeBrowser.storage.local.set({ apiKey: '' });
const apiKey = await getApiKey();
expect(apiKey).toBeNull(); // ✓ PASSED
```

### Case 3: Valid key stored
```typescript
await fakeBrowser.storage.local.set({ apiKey: 'sk-test-key-123' });
const apiKey = await getApiKey();
expect(apiKey).toBe('sk-test-key-123'); // ✓ PASSED
```

## Behavior
- Returns `null` (not undefined) when no key exists
- Returns `null` for empty string (graceful handling)
- Returns actual key value when present
- Never throws exceptions
